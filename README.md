📌 Overview

MediAssist is an AI-powered healthcare web application designed to simplify doctor appointment booking and provide basic medical assistance. The platform integrates intelligent features like symptom-based suggestions and chatbot interaction to improve user experience and accessibility in healthcare services.

🚀 Features
👨‍⚕️ Doctor Appointment Booking System
🤖 AI-based Symptom Checker
💬 Chatbot for basic medical guidance
📅 Appointment Scheduling & Management
🔐 Secure User Authentication
📊 User-friendly Dashboard
🧠 Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Python
Frameworks/Libraries: Flask / Django (based on your implementation)
AI/ML: NLP techniques, basic ML models
Database: MySQL / SQLite
🏗️ System Architecture
User interacts via web interface
Backend handles authentication & appointment logic
AI module processes symptoms and provides suggestions
Database stores user and appointment data
